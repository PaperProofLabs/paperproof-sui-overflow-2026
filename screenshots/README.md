# PaperProof Website Screenshots

This directory contains website screenshots captured from
`https://paperproof.site/` for documentation, presentations, and product
records.

The captures were taken in an automated Edge session connected through the
Chrome DevTools Protocol. Some captures show a connected browser wallet, but
wallet prompts and mainnet signing confirmations are not captured here.
